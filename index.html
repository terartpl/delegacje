Delegacje

Delegation system based on Symfony2 framework + bootstrap for mid-size and big companies - integration with SAP system

View on GitHub Download .zip Download .tar.gz

Aplikacja delegacje pozwala rozliczyć delegacje pracowników oraz przesłać dane do pliku.
1. Rozliczanie delegacji na podstawie realnych kosztów (brak diet)
2. Rozliczanie "kilometrówki"
3. Możliwość definiowania własnych rodzajów kosztów
4. Definiowanie oddziałów firmy
5. Eksport do pliku
6. Całkowicie za darmo

W wersji Enterprise dostępne są następujące funkcje:
1. Integracja z systemem SAP.
2. Workflow (obieg dokumentów) - pozwala ustalić strukturę i na jej podstawie przełożeni akceptują lub odrzucają delegacje.
3. Pokazywanie delegacji innych użytkowników. Jako użytkownik z takim uprawnieniem możesz przeglądać delegacje innych użytkowników.
4. Raporty oraz wykresy.
5. Filtrowanie delegacji. Możesz wyszykiwać delegacje na podstawie różnych parametrów.
6. Changelog (historia zmian). Dzięki tej opcji możesz śledzić wszystkie akcje wykonane na danej delegacji. Jeżeli włączona jest opcja "Workflow" to widoczne są również zatwierdzenia lub odrzucenia delegacji przez przełożonego wraz z komentarzami.

Delegation system allows your company to calculate all delegations of employees.

Installation

Download files and copy to destination folder.
Create empty database.
Open your command console and execute the following commands.
Create directory:

$ cd my_project_name/

$ mkdir app/cache

$ mkdir app/logs

$ mkdir web/logo
or if exist:
$ rm -rf app/cache/*

$ rm -rf app/logs/*
Setting up Permissions based on Symfony documentation:
See: Setting up Permissions section

Set permission for logo directory:
$ chmod 777 web/logo

Then run a composer update on command line:

$ php composer.phar update

Import tables:

$ mysql -u user_name -p db_name < delegations.sql

or you can also create the database tables schema from entities:


$ php app/console doctrine:schema:update --force

$ php app/console delegation:db:add-data
More on: Creating the Database Tables/Schema
Then install the required assets: 
$ php app/console assets:install

$ php app/console assetic:dump

Clear cache:

$ php app/console cache:clear --no-debug

Login to application:
link: your_localhost/login
login: admin
passwd: admin

Enterprise version

There is also enterprise version available with more features:
1. SAP integration.
2. Workflow which allows to set validator users who need to validate user delegations.
3. Show delegations of other users. You can select which user delegation you want to see. Option only available for users with special access right added by admin.
4. Reports and charts.
5. Delegation filters. You can select delegations based on different parameters.
6. Changelog. You can track all operations on delegation. Also you will see there acceptance or decline of delegation with the comment of the validator when workflow is enabled.
